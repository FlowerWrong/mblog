### nvm ###

```ruby
sudo apt-get remove --purge nodejs
curl https://raw.githubusercontent.com/creationix/nvm/v0.23.2/install.sh | bash
nvm ls-remote
nvm install 0.11
nvm use 0.11
nvm which 0.11
nvm ls
```

```ruby
npm update npm -g
npm install moduleName --save-dev  # 自动把模块和版本号添加到devdependencies部分
```
