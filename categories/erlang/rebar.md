## Rebar

#### Install [rebar](https://github.com/rebar/rebar/wiki/Getting-started)

```ruby
wget https://raw.github.com/wiki/rebar/rebar/rebar && chmod u+x rebar
sudo mv rebar /usr/local/bin
rebar -c
```
